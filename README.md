# Buddies OS

Buddies OS is a Next.js application with Supabase backend support.

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Jest

## Getting Started
1. Install dependencies:
   npm install

2. Start development server:
   npm run dev

3. Run lint:
   npm run lint

4. Run tests:
   npm test

## Environment
Create a local environment file before running the app:
- .env.local

## Notes
This repository excludes build artifacts, temp files, and generated service worker assets from version control.
